Que, porqué y para qué

¿ Que es sideladCodeStack ? SCStack una forja de Desarrollo, un ALM Tool para controlar el ciclo de vida de una aplicación. ¿ Por qué aparece ? Por la necesidad de centrar la atención en el desarrollo facilitando la composición del entorno estableciendo unas herramientas para guiar el proceso con buenas prácticas. ¿ Para que sirve ? Para ayudar a construir software dentro de un entorno controlado que facilita la evolución del mismo siguiendo unas pautas. El desarrollo de este proyecto se basa en el diseño e implementación del proceso de desarrollo acorde con las metodologías ágiles a través de la forja SidelabCode Stack. Unificando el comportamiento de las herramientas a través de las distintas APIs basadas en la interoperabilidad, facilitando la instalación, la replicación y la recuperación de los datos mediante el uso de Software Libre para construir Software de calidad.
